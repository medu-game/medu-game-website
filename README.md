# medu.game — platform website

Static, bilingual (NL/EN) marketing/overview site for the medu.game learning platform.

- `index.html` — platform overview (home)
- `modules/` — per-module detail pages (ABCDE, ALS, ECG, resuscitation & AED)
- `assets/` — CSS, JS, fonts, images, video

Plain static HTML/CSS/JS. No build step. Language toggle (NL/EN) is in `assets/medu-gallery.js`.
